# Jakarta EE: Component Model and Dependency Management
Проект: Сервис «Словарь пользователя» (Jakarta EE 10).

### Технологии:
* **EJB**: Stateless, Stateful, Singleton
* **CDI**: Qualifiers, Events, Observes
* **REST**: JAX-RS
* **Server**: Eclipse GlassFish 8.0.0
* **Build**: Maven (UTF-8 configured)
